# AI Resume & Portfolio Builder

This project generates personalized resume content using rule-based Generative AI
and Natural Language Generation (NLG).

## Technologies Used
- Python
- Google Colab
- Streamlit UI

## Features
- Dynamic resume generation
- User-specific input handling
- Professional summary creation

## Description
The system takes user inputs such as name, role, skills, education, and projects,
and generates structured resume content automatically using AI logic.
